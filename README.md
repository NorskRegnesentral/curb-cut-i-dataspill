## NO

### Generelt

Kan funksjoner laget for tilgjengelighet gi en bedre spillopplevelse for alle?

Curb cut-effekten i dataspill undersøker hvordan slike løsninger påvirker spillopplevelsen for personer med og uten funksjonsnedsettelser. Prosjektet kartlegger bruken og effekten av tilgjengelighetsfunksjoner, og utforsker hvordan prinsippene bak curb-cut-effekten kan bidra til mer inkluderende spilldesign.

Mer om prosjektet ligger [her](https://nr.no/prosjekter/dataspill-uten-barrierer-curb-cut/).

## EN

### General

How can features designed for inclusion improve gaming for everyone?

The Curb-Cut Effect in Gaming explores how these features impact the gaming experience for players with and without disabilities. The project maps how accessibility solutions are used, what impact they have, and how the principles behind the curb-cut effect can contribute to more inclusive game design.

More about the project can be found [here](https://nr.no/en/projects/gaming-without-barriers-curb-cut/)
